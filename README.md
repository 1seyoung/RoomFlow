# RoomFlow

사용자가 입력한 방 도면과 원하는 무드를 기반으로 AI Agent가 자동으로 가구를 배치하고 실제 제품을 추천하는 시스템 프로토타이핑

## 핵심 시나리오

### 사용자 플로우

1. 도면 생성
```
그리드 기반 픽셀 방식으로 방 구조 그리기
방1/방2/방3, 부엌, 화장실, 현관, 복도, 벽, 문, 창문 배치
현관은 항상 하단(남쪽) 고정
```

2. 스타일 입력 & AI 에이전트 동작
```
사용자가 자유 형식으로 원하는 스타일과 가구 입력
AI Agent가 입력을 분석하여 스타일 추출, 필요한 가구 결정, 최적 위치 자동 배치, 유사한 실제 제품 검색
```

3. 결과 확인
```
추천 제품 카드 표시
```


---

수행 업무 예시(목표)
- "미니멀한 북유럽 스타일로 꾸며줘"
- "미니멀한 화이트 톤 , 방1에 침대랑 책상 넣어줘"
- "북유럽 스타일로 방1: 퀸 침대, 원목 책상, 옷장 부엌: 4인 식탁 냉장고 전체적으로 화이트와 우드 톤"

## 기술적 도전 과제

1. 가구 배치 알고리즘

    - 충돌 감지
    - 동선 최적화
    - 공간 활용도 극대화


2. 자연어 파싱

    - 다양한 입력 형식 처리
    - 모호한 표현 해석


3. Vector 검색 정확도

    - 스타일 임베딩 품질
    - 필터링 로직
